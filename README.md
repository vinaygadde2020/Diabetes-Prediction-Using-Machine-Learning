# Diabetes-Prediction-Using-Machine-Learning

Machine Learning project for predicting diabetes using data preprocessing, exploratory data analysis (EDA), classification algorithms, model evaluation, and an interactive prediction system.

## 📌 Project Overview

This project aims to predict whether a patient is diabetic or non-diabetic using the Pima Indians Diabetes Dataset. The project covers the complete Machine Learning workflow, including data cleaning, exploratory data analysis, feature engineering, model training, hyperparameter tuning, and performance evaluation.

## 🎯 Objectives

* Analyze diabetes data and identify important features.
* Perform data cleaning and preprocessing.
* Conduct exploratory data analysis (EDA).
* Build and compare multiple machine learning models.
* Evaluate model performance using various metrics.
* Develop an interactive prediction system.

## 📂 Dataset Information

* Dataset: Pima Indians Diabetes Dataset
* Total Records: 768
* Features: 8 input features and 1 target variable
* Target Variable:

  * 0 → Not Diabetic
  * 1 → Diabetic

## 🔍 Key Findings

* Glucose is the strongest predictor of diabetes.
* BMI and Age significantly influence diabetes risk.
* Proper preprocessing and feature scaling improve model performance.
* Random Forest achieved the best overall performance among the implemented models.
